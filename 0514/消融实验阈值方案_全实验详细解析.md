## 10. 横向结论（按可落地性）

1. **最稳定的双90组合**：`S2-A90`（93.6% / 90.2%，base 81.4%）与 `S3-B90`（98.0% / 90.1%，base 87.6%）。  
2. **双95几乎不可迁移**：只有 `S2-A95` 在 test 仍满足双95，但 base 仅 10.5%，临床可用性很弱。  
3. **常见现象**：B 方案普遍能把 base 拉高到 85%~96%，但 low_p 常掉到 87%~89%。  
4. **val→test 偏移需单独监控**：如 D1、D3、S4 在 val 可选出的方案，test 上可能出现 high_p/high_R 下滑。
