## 🔬 实验详细配置与结果

### Exp#1: 基础双头联合训练
| 项目 | 配置 |
|------|------|
| **核心思路** | SwinV2 encoder + UNet decoder + GAP cls head, 联合训练 |
| **输入** | 3ch RGB (mask变为分割目标) |
| **分割** | 2类, CE+Dice, lesion权重=5.0 |
| **分类** | lambda_cls=1.0 |
| **Epochs** | 100 |
| **结果** | F1=0.6157, 阈值优化=0.6215 (thresh=0.450) |
| **分割Dice** | 0.8643 |
| **问题** | 严重过拟合 (Train 99% vs Test 72%); 丢失mask输入信息 |

---

### Exp#2: 多尺度分类头 + Mixup
| 项目 | 配置 |
|------|------|
| **核心思路** | Exp#1 + 多尺度分类头 + Mixup/CutMix |
| **分类头** | 多尺度 GAP(f3+d3+d2+d1) → MLP |
| **Mixup** | img+mask同步混合 |
| **lambda_cls** | 3.0 |
| **结果** | F1=0.5905 (Epoch 10), 阈值优化=0.6083 (thresh=0.580) |
| **问题** | **Mixup与分割冲突!** F1剧烈震荡, 分割Dice只有0.83 |

**核心教训**: Mixup/CutMix 对分割+分类联合训练**有害**！混合mask破坏语义。

---

### Exp#3: 4ch输入 + 分割辅助正则化
| 项目 | 配置 |
|------|------|
| **核心思路** | 恢复4ch输入(RGB+mask), 分割作为辅助正则化 |
| **输入** | 4ch (RGB + lesion_mask) |
| **分类头** | GAP → MLP |
| **lambda_cls** | 2.0 (分类为主) |
| **Mixup** | 无 |
| **Epochs** | 100 |
| **结果** | F1=0.6098, 阈值优化=0.6161 (thresh=0.505) |
| **分割Dice** | 0.9939 |
| **问题** | GAP分类头太简单, 没有利用分割特征 |

---

### Exp#4: Seg-Guided Attention + Metadata 🏆
| 项目 | 配置 |
|------|------|
| **核心思路** | 用分割概率图引导分类关注病灶区域, 融合临床metadata |
| **输入** | 4ch (RGB + lesion_mask) |
| **分类头** | Seg-Guided Attention → MLP + Metadata融合 |
| **Metadata** | age, gender, size, flow, morph (6维) |
| **lambda_cls** | 2.0 |
| **Dropout** | cls=0.4 |
| **Epochs** | 100 |
| **结果** | **F1=0.6707**, 阈值优化=0.6741 (thresh=0.505) |
| **分割Dice** | 0.9851 |
| **benign召回** | 0.5116 (显著提升!) |

```
              precision    recall  f1-score   support
      benign     0.5000    0.5116    0.5057       129
    no_tumor     0.8389    0.8325    0.8357       394
   macro avg     0.6694    0.6721    0.6707       523
```

**成功因素**: Seg-Guided Attention + Metadata融合 + 4ch输入 + 无Mixup

## 📈 关键发现与教训

### ✅ 已验证的有效策略

| 策略 | 实验依据 | 效果 |
|------|---------|------|
| **4ch输入 (RGB+mask)** | Exp#1 vs Exp#3 | 保留mask输入信息至关重要 |
| **Seg-Guided Attention** | Exp#4 vs Exp#3 | +0.0609 F1提升 |
| **Metadata融合** | Exp#4 vs Exp#6 | +0.0624 F1提升 |
| **lambda_cls=2.0** | 全部实验 | 分类为主、分割为辅 |
| **无Mixup** | Exp#2教训 | Mixup与分割冲突 |
| **适度正则化** | Exp#7 | DropPath+大WD有效 |

### ❌ 已验证的无效/有害策略

| 策略 | 实验依据 | 教训 |
|------|---------|------|
| **Mixup/CutMix** | Exp#2 | 混合mask破坏语义 |
| **更高分辨率320** | Exp#5/6 vs Exp#3/4 | 分辨率不是瓶颈 |
| **MaxViT backbone** | Exp#5/6 | 在此任务上不如SwinV2 |
| **Seg-Guided Attn无meta** | Exp#6 | 需要配合metadata |
| **SupConLoss** | Exp#9 | 未带来提升 |

### ⚠️ 过拟合观察

所有SwinV2实验在**Epoch 10左右**达到最优F1, 之后持续下降:
- Exp#3: Best@Epoch10, 之后F1从0.61下降到0.58
- Exp#4: Best@Epoch10, 之后F1从0.67下降到0.59
- Exp#7: Best@Epoch10, 50 epochs足够

**建议**: 
1. 训练30-50 epochs即可
2. 使用Early Stopping
3. 增加正则化 (DropPath, Dropout, WeightDecay)

## 🎯 最佳配置推荐

基于所有实验结果, **Exp#4** 是最佳配置:

```python
# 最佳配置
backbone = "SwinV2-Tiny"
image_size = 256
input_channels = 4  # RGB + lesion_mask

# 分割
seg_decoder = "UNet with skip connections"
seg_loss = "CE + Dice"
lesion_weight = 5.0

# 分类  
cls_head = "Seg-Guided Attention + MLP"
metadata = ["age", "gender", "size", "flow", "morph"]
lambda_cls = 2.0
cls_dropout = 0.4

# 训练
epochs = 30-50  # 不需要100
lr_backbone = 2e-5
lr_head = 2e-4
weight_decay = 0.05-0.1
warmup_epochs = 5-8
mixup = False  # 重要!

# 可选正则化
drop_path_rate = 0.2-0.3
ema = 0.999
```
