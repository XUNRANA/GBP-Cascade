## 实验改进概况（4-5句）

0502 这一轮主要围绕 baseline 的三类问题做改造：过度手术、良性分数发散、校准两极化。  
其中 B（校准路线）提升最大：高风险精确率和校准误差显著优化，但把大量 benign 压到了 low。  
0510 的 E/F/G/H/I 主要在修复这个 benign→low 过高的临床风险。  
结果是多头/级联方案（尤其 H/I）明显降低了 benign→low，但 no_tumor→low 同时大幅下降。  
目前形成两条路线：B 更偏“节流效率”，H/I 更偏“临床谨慎”，尚无单一方案同时最优。

## 每个实验主要改进点（1句话版）

| 实验 | 主要改进点 |
|---|---|
| 0502-Baseline | 三模态融合 + ordinal 风险回归（1.0/0.5/0.0）+ 双阈值分层，作为后续全部实验对照基线。 |
| 0502-A（SupCon） | 在 fused 特征后增加投影头与监督对比学习，目标是拉开 benign/no_tumor 的特征边界。 |
| 0502-B（Calib） | 改为非对称 target（1.0/0.35/0.0）并加入早停与温度校准，主攻过拟合与概率校准。 |
| 0502-C（TwoStage） | 增加 benign-vs-no_tumor 的二阶段 head，尝试把“恶性识别”和“非恶性细分”解耦。 |
| 0502-D（Modal） | 引入模态 dropout、gate 熵正则、文本 MLM 和 meta noise，提升跨分布稳定性。 |
| 0510-E（Aux-Cls） | 加三分类辅助头（mal/ben/nt）和先验路由决策，尝试强制 benign 不落 low。 |
| 0510-F（Triple-Head） | 将单 risk head 改成 high/med/low 三个独立 head，用 argmax 直接做风险分层。 |
| 0510-G（Margin） | 在 baseline 上加入排序间隔损失（nt < ben < mal），强化三类分数相对次序。 |

| 0510-I（B+H Fusion） | 在 H 基础上叠加 B 风格 ordinal 辅助监督，试图同时兼顾分离能力与校准收益。 |

## Baseline 重点解析（5句）

Baseline 的核心优势是安全性强：mal→low 基本守住，high_recall 长期较高。  
它的主要短板是 high_precision 不稳，在 no_tumor 占比升高时更容易把非恶性推到 high。  
从分数分布看，baseline 存在明显校准两极化，中间风险段利用不充分，阈值鲁棒性较弱。  
训练上还存在后期过拟合迹象：最佳泛化通常出现在较早 epoch，继续训练反而恶化校准与稳定性。  
后续全部实验本质都在修这两件事：一是概率校准/阈值稳定，二是 benign 与 no_tumor 的可分性。

## 数据集、训练数据、训练方法、结果（概况）

### 1) 数据集是怎么样的

任务是三分类（malignant / benign / no_tumor）并映射到三档风险（high / medium / low）。  
本轮核心用到两个测试分布：Test-111 为 1:1:1，Test-112 为 1:1:2（no_tumor 翻倍）用于稳定性检验。  
在 Test-112 设定下，总样本数通常为 456（114 / 114 / 228）。

### 2) 训练数据是啥

主训练数据来自 `0502dataset111`（约 1:1:1）。  
训练时采用 patient-level 分组划分 train/val（val 约 15%），避免同一患者信息泄漏。  
`0502dataset112`主要作为外部分布测试，不参与主训练。

### 3) 训练方法是啥

主干是三模态融合：SwinV2-Tiny（4通道 RGB+mask）+ Seg-Guided Attention + 冻结BERT文本分支 + 10维临床特征 + Gated Fusion。  
优化策略包括 BalancedBatchSampler（3:3:2）、AMP、梯度裁剪、验证集阈值搜索。  
实验 A~I 在此基础上分别尝试 SupCon、校准（非对称target+早停+温度）、TwoStage、Modal Dropout、Aux-Cls、Triple-Head、Margin、Cascade、Fusion。

### 4) 得到的结果怎么样

最强“节流”方案是 0502-B（Calib）：high_precision 与校准显著提升，no_tumor→low 很高。  
但 B 的副作用是 benign→low 偏高（大量良性被压到 low）。  
0510-H/I 明显改善 benign→low（可降到 2.6%/0%），但 no_tumor→low 同时大幅下降。  
结论是目前仍是权衡：B 更偏效率，H/I 更偏谨慎，尚无单模型同时最优覆盖全部目标。
