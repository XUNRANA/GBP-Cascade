# 特征可视化实验总结

> **目标模型**: `20260414_task3_SwinV2Tiny_segcls_3` (best 三分类模型, Softmax Macro-F1)
> **测试集**: 456 张超声图像, 类别 = malignant (114) / benign (114) / no_tumor (228)
> **输出根目录**: `logs/feature_analysis_best_softmax_segcls3/`

实验分两大块:
- **Phase 1** — 深度特征多视角分析 (8 个隐藏层的 t-SNE / 3 种降维对比 / 9 属性上色 / 距离分布 / 判别维度 / 灰区样本)
- **Phase 2** — PyRadiomics 107 维经典影像组学特征 (提取 + 降维 + 单特征统计 + biomarker 可视化)

---

## Phase 1. 深度特征多视角分析

输出: `logs/feature_analysis_best_softmax_segcls3/deep/`

### 图 1.1 — 多层 t-SNE 网格 (`1.1_multilayer_tsne_grid.png`)

**怎么画**: 对模型 8 个中间特征分别做 t-SNE, 4×2 网格排开, 标题标注每层的 silhouette.

| 位置 | 维度 | silhouette | 含义 |
|---|---|---|---|
| backbone f0 | 96 | 0.009 | 浅层纹理, 三类严重混叠 |
| backbone f1 | 192 | 0.015 | 略好, 但仍几乎随机 |
| backbone f2 | 384 | 0.036 | 开始出现结构 |
| backbone f3 | 768 | 0.018 | 高维过度压缩, 反而回落 |
| cls_proj (f2→256, 无 attn) | 256 | 0.081 | 显著跳跃 |
| cls_feat_img (seg-attn pool) | 256 | 0.106 | seg-attention 的增益 +0.03 |
| **cls_feat_320 (fused)** | 320 | **0.115** | 与既有既有报告一致 ✓ |
| penultimate (cls_head[0] 后) | 128 | **0.182** | 最分得开 |

**解读**:
- silhouette 随深度**近单调上升**, 符合表征学习直觉.
- f3 回落是 Swin 末层特征更抽象/更任务无关的典型现象, `cls_proj` 之后的 1×1 conv + seg-guided attention 把任务相关信号重新聚焦.
- **真正把三类分开的是 penultimate 层**, 即分类头里的 `Linear(320→128)+GELU`. 这意味着 `cls_feat_320` 虽可视化已较好, 但仍携带冗余.

### 图 1.2 — PCA vs t-SNE vs UMAP 对比 (`1.2_pca_tsne_umap.png`)

**怎么画**: 在 320-D cls_feat 上并排三张散点图, 每张标题写方法 + silhouette.

| 方法 | silhouette | ARI (KMeans-3) |
|---|---|---|
| PCA-2D | 0.164 | — |
| t-SNE-2D | 0.164 | 0.282 |
| UMAP-2D | 0.155 | — |

**解读**:
- 三种降维给出高度一致的布局 → 说明类别结构是数据内禀而非方法伪影.
- PCA 前两主成分就占 49.8% 的方差, 意味着特征空间的主要变化方向就是分类相关的方向, 不存在"被噪声主导"的问题.
- malignant 明显独立, benign↔no_tumor 部分重叠 (与 1.1 同结论).

### 图 1.3 — 多属性着色 3×3 网格 (`1.3_attribute_color_grid.png`)

**怎么画**: 同一张 t-SNE 2D 坐标换 9 种颜色:

| 行\列 | 1 | 2 | 3 |
|---|---|---|---|
| 分类 | 真实类 | 预测类 | 正确/错误 |
| 概率 | ordinal score (0-1) | softmax max prob | 预测熵 |
| 临床 | size_bin | flow_bin | morph_bin |

**解读**:
- **第一行** (真实 vs 预测 vs 正确性): 直接看模型决策边界; 错误样本几乎全集中在 ben↔no 的过渡区.
- **第二行 ordinal score**: 连续色带从 malignant 的 1.0 渐变到 no_tumor 的 0.0, 呈现自然"风险梯度", benign 卡在 0.5 左右 → 与模型设计的 ordinal target [1, 0.5, 0] 吻合.
- **第二行熵**: 高熵点**精准坐落在 ben↔no 边界**, 是后续挑"灰区样本复核"的依据.
- **第三行临床**: 例如 size_bin 大的点集中在 malignant 簇, flow_bin 有血流的点集中在肿瘤类簇, morph_bin 不规则形状的点集中在 malignant — 验证临床先验与深度特征一致.

### 图 1.4 — 类间/类内距离分布 (`1.4_distance_distributions.png`)

**怎么画**: 四联图: Violin (类内距离) / Boxplot (6 类对样本距离) / 质心矩阵 heatmap / 样本 silhouette 堆叠直方图.

**数值**:
- 类内平均距离: mal=4.74, ben=5.16, no=4.71
- 质心距离: mal-ben=8.76, mal-no=9.85, **ben-no=1.92** ← 瓶颈
- 每样本 silhouette 直方图: benign 的众数接近 0, 意味着很多 benign 样本"不比归到 no_tumor 差"

**解读**:
- 恶性类被**强力**推开了 (质心距离 8-10), 但 ben vs no 的质心距离只有 1.92 —— 还不到类内半径的一半, 几何上本来就不可能用超平面干净切开.
- 这是全流程最大瓶颈, 也是 Phase 2 转向 PyRadiomics 找 ben-vs-no biomarker 的动机.

### 图 1.5 — 判别性维度分析 (3 张)

#### `1.5_top_dims_bar.png`
320 维中每一维独立做 3 类 ANOVA F 检验, 排序后绘制 top-30 维度的 F 值柱状图.

- 最强的 4 维 F 值 > 400 (dim #178=471, #312=418, #109=416, #28=415), 单维就能达到非常高的统计显著.
- p 值全部为 0 (下溢).

#### `1.5_top6_density_kde.png`
top-6 维度在三类上的 KDE 叠加 (每面板三条曲线).

- 明显看到某些维度是 "malignant detector" (在 mal 上高, ben/no 几乎重合)
- 另一些维度才在 benign 与 no_tumor 之间有分离
- 这直接说明 — 网络已经把"恶性 vs 非恶性"和"息肉 vs 无瘤"分配到**不同的坐标方向**

#### `1.5_top20_corr_heatmap.png`
top-20 判别维度之间的 Pearson 相关矩阵.

- 平均 |corr| = **0.81** → top 判别维度之间高度相关
- 意味着网络内部有很大冗余, 真正的"独立判别方向"可能只有 3-5 个
- 这对未来做特征压缩 / 蒸馏 / 可解释性是个切入点

### 图 1.6 — 灰区样本定位 (`1.6_confusion_zone_overlay.png`)

**怎么画**: t-SNE 2D 按 ordinal score 连续上色, 叠加两条阈值等值线 (t1=0.33, t2=0.66) 把空间划为绿灯/黄灯/红灯三段; 用不同 marker 高亮 malignant→no_tumor 漏诊样本 + 高置信误分类.

**数值**:
- `mal_miss_count = 2` (最严重的临床红线: 把恶性判成"无瘤")
- `hi_conf_wrong_count = 17` (模型 prob > 0.7 但依然错判)

**解读**:
- 2 个漏诊点的位置告诉我们哪些"伪装型 malignant"需要额外关注 (回到图像本身能定位具体病例).
- 17 个高置信误分类是"知识盲区", 适合加入难样本池.
- 绿/黄/红三段对应最终临床决策建议, 和 `实验计划.md §4.3` 的临床三段式一一对应.

## Phase 2. PyRadiomics 经典影像组学

输出: `logs/feature_analysis_best_softmax_segcls3/radiomics/`

### 特征提取 (`features_test_gb_roi.csv` / `features_test_lesion_roi.csv`)

提取 102 维 (first-order + shape2D + glcm + glrlm + glszm + gldm + ngtdm):

| ROI | 成功 / 总数 | 耗时 | 用途 |
|---|---|---|---|
| 胆囊 ROI (rectangle) | 453 / 456 | 250 s | 三类共同对比 |
| 病灶 ROI (polygon) | 342 / 342 | — | ben+no 内部对比 |

3 张失败均因 SimpleITK 强度归一化后 mask 为空 (`extract_log.json` 已记录).

### 图 2.3_pca_scree.png — PCA 累计方差曲线

- 前 2 主成分解释 **49.9%** 方差, **6 个 PC** 达到 80%, **10 个 PC** 达到 90%
- 意味着 102 维 PyRadiomics 特征的内禀维度大约是 10 维 → 绝大部分信号被少数几个方向承载

### 图 2.3_pca_2d.png — PCA 2D 散点

- sil = 0.001, 三类几乎完全重叠
- PC1 承载 29.7% 方差, 主要反映 "胆囊大小 / 对比度" 这种与类别无关的尺度

### 图 2.3_tsne.png — t-SNE 2D

- sil = **-0.020** (负值!), 即局部邻居里异类比同类还多
- 直接证明: **单独用经典影像组学几乎无法端到端区分三类**

### 图 2.3_umap.png — UMAP 2D

- sil = -0.014, 与 t-SNE 一致

**对比深度特征**: Phase 1 相同 2D silhouette ≈ 0.11-0.18; 经典特征 ≈ 0. 这 ~100 倍的差距清晰地说明 DL 学到了经典手工特征无法直接表达的组合.

### 图 2.5_volcano_3panels.png — Volcano 火山图

三面板 (mal-ben / mal-no / ben-no), x = Cliff's δ, y = -log10(q-FDR), q<0.05 的点用深红, 无显著性灰色, 每面板标注 top-8 最大效应量点.

| 比较 | q<0.05 显著数 / 102 |
|---|---|
| 3 类 ANOVA | 85 |
| malignant vs benign | 67 |
| malignant vs no_tumor | 79 |
| **benign vs no_tumor** | **41** ← 用户最关心 |

**解读**: 即便整体 2D 几何不可分, 单维度上仍有大量特征能区分 ben vs no. 这正是 biomarker 的价值所在 — 不是用它做端到端分类器, 而是用它做可解释指标.

### 图 2.5_top10_boxplot_grid.png — Top-10 按 ANOVA 箱线 + 散点

2×5 网格, 每格按 q_anova 从小到大排的前 10 个特征, 箱体 + jitter 散点.

**最强全局判别是 `ngtdm_Coarseness`** (粗糙度, 反映邻域灰度落差幅度), F=57.5, q=6e-21. 从箱线可见: malignant 明显低于 ben/no (malignant 组织更细腻?).

其它进入 top-10 的: glrlm_GrayLevelNonUniformity, glrlm_RunEntropy, glszm_SizeZoneNonUniformity, glcm_Imc1 等, **全部来自纹理族**, shape/firstorder 未进前 10 — 形态和一阶强度区分不动三类.

### 图 2.5_top5_violin_ben_vs_no.png — Benign vs No_tumor 最显著 5 个

Top 5 (按 q_ben_no 升序, 均 q < 5e-4):

| rank | feature | Cliff's δ | 方向 |
|---|---|---|---|
| 1 | glcm_DifferenceVariance | +0.31 | ben > no |
| 2 | glszm_SmallAreaEmphasis | +0.29 | ben > no |
| 3 | glszm_SizeZoneNonUniformity | +0.29 | ben > no |
| 4 | glszm_SizeZoneNonUniformityNormalized | +0.28 | ben > no |
| 5 | glcm_Correlation | -0.28 | no > ben |

**临床解读**: 良性胆囊息肉相较无肿瘤胆囊 → 纹理更不均匀 (DifferenceVariance / SizeZoneNonUniformity 更高, Correlation 更低) + 包含更多小面积灰度区 (SmallAreaEmphasis 更高). 这与病理直觉一致: **息肉是有内部结构的组织, 正常胆囊腔内主要是液性均质回声**.

### 图 2.5_top20_corr_heatmap.png — Top-20 Pearson 相关

- glcm 家族 (Contrast / DifferenceVariance / Imc1) 内部高度正相关
- glszm 家族 (SizeZoneNonUniformity* / SmallAreaEmphasis / ZonePercentage) 构成另一个高相关块
- 两大块之间也有 0.5-0.7 的相关
- **建模建议**: 106 维里真正独立的信号大约 5-8 维; LASSO 或 top-k PCA 压缩即足够

### 图 2.5_class_radar.png — 三类 Top-6 雷达

三类各自在 top-6 全局 biomarker 上的 z-score 均值画在 polar.

- malignant: 除了 Coarseness 方向明显凹陷, 其它维度接近 0
- no_tumor: 多个纹理维度明显偏低 (更均质)
- benign: 多个纹理维度偏高 (最不均质)

→ **纹理异质性 — benign > malignant > no_tumor** 的清晰排序, 可作为临床辅助结论.

### 图 2.6_gb_vs_lesion_compare.png — 胆囊 ROI vs 病灶 ROI

限定到 benign + no_tumor 342 张 (都有 lesion polygon), 对比两种 ROI 下 102 维特征的 ben-vs-no 分离能力.

| ROI | n | sil(t-SNE) | sil(raw) |
|---|---|---|---|
| gallbladder rect | 339 | 0.013 | 0.021 |
| **lesion polygon** | 342 | **0.028** | **0.040** |

→ **病灶 ROI 的可分性约是胆囊 ROI 的 2 倍**. 未来若要为 ben-vs-no 做专门分类器, 应以 lesion mask 为主 ROI; 主模型选择 gb ROI 是因为无瘤类没有 lesion polygon, 要对全 456 张做统一提取的工程权衡.

## 关键结论 (给论文用)

1. **深度特征演化**: silhouette 在 8 个层级近单调上升 (0.009 → 0.182), 主要跳跃来自 seg-guided attention (cls_proj → cls_feat_img, +0.03) 和最末的 Linear+GELU (cls_feat_320 → penultimate, +0.07).

2. **三分类的瓶颈是 benign↔no_tumor**: 质心欧氏距离仅 1.92, 类内半径 4.7-5.2, 几何上本就不可能完全分开.

3. **经典影像组学单独不够**: 102 维 PyRadiomics 在 2D 降维下 silhouette ≈ 0, 但 FDR-显著单特征达 85 个 (3 类 ANOVA) / 41 个 (ben-vs-no); 价值在 biomarker 可解释性, 不在端到端分类.

4. **Top Biomarker 指向纹理异质性**: 良性息肉相对无瘤胆囊表现为更不均匀的纹理 (↑Contrast/DifferenceVariance/SmallAreaEmphasis, ↓Correlation), 与"息肉是有结构的组织"的病理直觉一致.

5. **ROI 选择建议**: 若专做 ben-vs-no 的经典特征分类器, lesion polygon 优于 gallbladder rectangle (silhouette 翻倍).

6. **冗余诊断**: top 判别维度互相相关 |r|≈0.81 (深度) / glcm-glszm 高相关 (经典), 说明未来可以用小得多的特征子集复现本实验.
