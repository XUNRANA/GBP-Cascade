# 对比损失消融实验总结（SupCon）

> 更新日期：2026-04-20  
> 实验目标：验证 `SupCon 对比损失` 对三分类任务（malignant / benign / no_tumor）的实际贡献。

### 1.3 说明

## 2. 核心指标对比（最终测试）

| 指标 | 有 SupCon | 无 SupCon | 差值（无-有） |
|---|---:|---:|---:|
| Best Val F1 | 0.7347 | **0.7460** | +0.0113 |
| Softmax Test Macro-F1 | 0.6857 | **0.6938** | +0.0081 |
| Softmax Test Acc | 0.7061 | **0.7237** | +0.0176 |
| Ordinal(最优阈值) Test Macro-F1 | 0.6971 | **0.7010** | +0.0039 |
| Ordinal(最优阈值) Test Acc | 0.6952 | **0.7061** | +0.0109 |
| Seg Dice | **0.8497** | 0.8024 | -0.0473 |

> 观察：无 SupCon 在“分类主指标”上略优，但分割 Dice 明显下降。

---

## 5. 结论

1. 从纯分类主指标看，**无 SupCon** 略优（Softmax/Ordinal 的 Macro-F1 和 Acc 都有小幅提升）。  
2. 从临床安全与类别均衡看，**有 SupCon** 更稳：  
   - malignant sensitivity 更高（0.9123 vs 0.8684）  
   - benign 识别率更高（0.5175 vs 0.4737）  
   - Seg Dice 明显更好（0.8497 vs 0.8024）  
3. 当前结果显示：SupCon 并未显著提高总体 F1，但对“安全性与 benign/no_tumor 平衡”仍有正作用。

---

## 6. 建议

1. 若优先目标是 `Macro-F1/Acc`，可继续沿无 SupCon 路线微调。  
2. 若优先目标是临床安全（尤其 malignant sensitivity、benign recall），建议保留 SupCon。  
3. 下一步可做 `lambda_con` 网格（如 `0.0 / 0.1 / 0.2 / 0.3`），寻找折中点，而不是二选一。
