# 0417 临床约束下最大化无肿瘤识别 — Exp#19 系列实验计划

## 一、临床定位与目标

### 1.1 本次任务的临床本质

在 `task_2`（benign vs no_tumor 二分类）中：

- **benign（真肿瘤 / 需手术）**：129 例 (test)，模型不能漏（漏 = 漏诊恶性倾向，严重临床后果）
- **no_tumor（普通息肉 / 免手术）**：394 例 (test)，模型应尽量挑出（挑出 = 避免防御性手术）

临床医生真正关心的指标不是 F1、也不是整体准确率，而是：

> **在 benign 召回 ≥ 90% / 95% 的硬约束下，究竟能安全地识别出多少 no_tumor 免挨一刀？**

换成统计语言：**最大化 TPR@(FPR≤0.10 / FPR≤0.05)**，即 ROC 曲线高敏感度端的命中数。

### 1.2 成功标准

| 场景 | 当前最好 (Exp#18-3) | 本次目标 |
|---|---|---|
| benign 召回 ≥ 90% → no_tumor 命中 | 134 / 394 (34.0%) | **≥ 170 / 394 (43%)** |
| benign 召回 ≥ 95% → no_tumor 命中 | 83 / 394 (21.1%) | **≥ 115 / 394 (29%)** |
| pAUC@FPR≤0.05 | 0.146 | **≥ 0.22** |
| ECE（校准误差） | 0.32 | **< 0.08** |

### 2.2 五点核心诊断

1. **ROC 天花板不高**：AUC 0.67–0.73，单纯调阈值空间有限，要**抬起整条 ROC**
2. **训练目标与临床目标脱节**：Macro F1 选模型 → 中段准，两端（高敏感度区）没被优化
3. **概率校准极差**：ECE=0.32（Exp#18-3）→ 阈值对应的实际召回率与期望值偏差巨大
4. **阈值选在 test 上** → 乐观偏差，真实部署会差一截
5. **多模型结果有差异化**：Exp#18（高AUC）+ Exp#18-3（高pAUC）互补，集成可免费 +14 no_tumor（已实测）

### 6.3 Exp#19-1 ＝ 18-3 骨架 + pAUC Surrogate Loss

**唯一改动**：新增 pAUC loss 项

**新增组件** `PAUCSurrogateLoss`：
```python
class PAUCSurrogateLoss(nn.Module):
    """Top-K pairwise hinge on prob_benign.
    Penalty for benign_i being ranked BELOW no_tumor_j.
    """
    def __init__(self, K=4, margin=0.1):
        super().__init__()
        self.K, self.margin = K, margin

    def forward(self, cls_logits, labels):
        # labels: 0=benign, 1=no_tumor
        p_benign = cls_logits.softmax(-1)[:, 0]
        benign_mask  = labels == 0
        nt_mask      = labels == 1
        if benign_mask.sum() == 0 or nt_mask.sum() == 0:
            return cls_logits.new_zeros(())
        p_b  = p_benign[benign_mask]
        p_nt = p_benign[nt_mask]
        K = min(self.K, p_b.numel())
        hard_b = p_b.topk(K, largest=False).values  # 最难的 K 个 benign (prob 最低)
        # 全 pair hinge
        diff = self.margin - (hard_b[:, None] - p_nt[None, :])
        return diff.clamp_min(0).mean()
```

**损失分阶段混合**：
- Stage 1 [epochs 1–40]: `L = L_segcls`（纯 curriculum 预热, 先学 benign-prior）
- Stage 2 [41–80]: `L = L_segcls + 0.5 × L_pauc`
- Stage 3 [81–100]: `L = L_segcls + 1.0 × L_pauc`

**目的**：单独量化"训练目标与临床目标对齐"的贡献，pAUC@0.05 预期 0.146 → 0.18–0.20。

**GPU**: 0，预计墙钟 2.5h。

### 6.4 Exp#19-2 ＝ 19-1 + SAM + SWA + Temperature Scaling

**新增组件 1：SAM (Sharpness-Aware Minimization)**
```python
class SAM(torch.optim.Optimizer):
    def __init__(self, params, base_optimizer, rho=0.05, **kwargs):
        defaults = dict(rho=rho, **kwargs)
        super().__init__(params, defaults)
        self.base_optimizer = base_optimizer(self.param_groups, **kwargs)

    @torch.no_grad()
    def first_step(self, zero_grad=False):
        grad_norm = self._grad_norm()
        for group in self.param_groups:
            scale = group["rho"] / (grad_norm + 1e-12)
            for p in group["params"]:
                if p.grad is None: continue
                e_w = p.grad * scale.to(p)
                p.add_(e_w); self.state[p]["e_w"] = e_w
        if zero_grad: self.zero_grad()

    @torch.no_grad()
    def second_step(self, zero_grad=False):
        for group in self.param_groups:
            for p in group["params"]:
                if p.grad is None: continue
                p.sub_(self.state[p]["e_w"])
        self.base_optimizer.step()
        if zero_grad: self.zero_grad()
    # ... _grad_norm() 省略
```

训练循环改造（`train_one_epoch_text_sam`）：
```python
# forward 1
loss_1 = compute_loss(model(batch))
loss_1.backward()
optim.first_step(zero_grad=True)
# forward 2 (on perturbed weights)
loss_2 = compute_loss(model(batch))
loss_2.backward()
optim.second_step(zero_grad=True)
```

**关键：AMP 不能与 SAM 共用 GradScaler**（double-backward 冲突）→ 改为只用 `autocast` 不用 scaler。

**新增组件 2：SWA Wrapper**
- 最后 20 epoch (epochs 81–100) 做权重平均：`torch.optim.swa_utils.AveragedModel`
- 训练结束后 `torch.optim.swa_utils.update_bn(train_loader, swa_model)`（BN 统计量校正）
- 同时评估原模型和 SWA 模型，取 dev pAUC@0.05 高者

**新增组件 3：Temperature Scaler**
```python
class TemperatureScaler(nn.Module):
    def __init__(self):
        super().__init__()
        self.T = nn.Parameter(torch.ones(1))

    def forward(self, logits):
        return logits / self.T

    def fit(self, dev_logits, dev_labels, max_iter=200):
        self.T.data.fill_(1.0)
        opt = torch.optim.LBFGS([self.T], lr=0.01, max_iter=max_iter)
        def closure():
            opt.zero_grad()
            loss = F.cross_entropy(dev_logits / self.T, dev_labels)
            loss.backward()
            return loss
        opt.step(closure)
        return self.T.item()
```

**回退方案**：若 SAM 发散（常见于 AMP 关闭后数值波动）或总时间 >4h，退化为仅保留 SWA + TempScale。

**目的**：量化"泛化 + 校准"贡献，ECE 从 ~0.25 → <0.08，pAUC@0.05 +0.02–0.03。

**GPU**: 1，预计墙钟 3.5h（SAM ≈ 1.4× 单次训练）。

`between_class_mixup_fused(fused, labels, alpha, beta)`：
- `λ ~ Beta(α=2.0, β=0.5)`（偏向第一个样本，避免过强扰动）
- 每个样本 i 随机抽 `j ≠ i` 且 `labels[j] ≠ labels[i]`（严格跨类）
- `fused_mix[i] = λ·fused[i] + (1-λ)·fused[j]`
- `labels_mix`：soft label `[λ onehot_i + (1-λ) onehot_j]`，分类 loss 改 soft CE

**只在 Stage 2 [41–80] 开启**，避免 Stage 1 curriculum benign-prior 学习被干扰，Stage 3 回到纯样本巩固。seg 头完全不 mixup（保分割质量）。

### 10.1 主指标与次指标

| 主指标 | 定义 | 目标 |
|---|---|---|
| pAUC@FPR≤0.05 | `roc_auc_score(y, prob_nt, max_fpr=0.05)` | ≥ 0.22 |
| no_tumor@良≥95% | `find_constrained_threshold(miss_rate=0.05).no_tumor_recall × 394` | ≥ 115 |
| no_tumor@良≥90% | `find_constrained_threshold(miss_rate=0.10).no_tumor_recall × 394` | ≥ 170 |

| 次指标 | 定义 | 目标 |
|---|---|---|
| ECE | 10-bin expected calibration error | < 0.08 |
| Brier | `mean((prob_benign - (label==0))**2)` | < 0.20 |
| TPR@FPR=0.02 | 更严格的漏诊约束点 | > 0.10 |

### 10.2 阈值选择的铁律

1. **所有阈值必须在 dev 上搜索**，test 上只评估一次
2. 禁止使用 `test` 反向挑 `miss_rate`；所有 miss_rate∈{0.02, 0.05, 0.10} 统一在 dev 上搜，test 上读数
3. Phase A 的"test 上 5-fold 校准"作为独立诊断，不参与最终决策

## 附录 A：关键公式

### A.1 pAUC Surrogate

$$\mathcal{L}_{\text{pAUC}} = \frac{1}{|\mathcal{B}_K||\mathcal{N}|} \sum_{i \in \mathcal{B}_K} \sum_{j \in \mathcal{N}} \max\!\Big(0,\ \gamma - (p^{b}_i - p^{b}_j)\Big)$$

其中 $\mathcal{B}_K$ 为 batch 中 $p^{b}$（benign 概率）最低的 K 个 benign 样本（即"最易漏诊"的 benign），$\mathcal{N}$ 为 batch 中所有 no_tumor 样本，$\gamma=0.1$ 为 margin。

### A.2 Temperature Scaling

$$\hat T = \arg\min_T \ \mathrm{NLL}\big(\mathrm{softmax}(z_{\mathrm{dev}} / T),\ y_{\mathrm{dev}}\big)$$

推理：`probs = softmax(z_test / T_hat)`。

### A.3 Between-class Mixup on Fused

$$\lambda \sim \mathrm{Beta}(2.0, 0.5),\quad \tilde f_i = \lambda f_i + (1-\lambda) f_j,\quad \tilde y_i = \lambda y_i + (1-\lambda) y_j,\ \mathrm{where}\ y_j \ne y_i$$

### A.4 ECE

$$\mathrm{ECE} = \sum_{m=1}^{M} \frac{|B_m|}{N} \Big| \mathrm{acc}(B_m) - \mathrm{conf}(B_m) \Big|$$

M=10 等宽 bin。
