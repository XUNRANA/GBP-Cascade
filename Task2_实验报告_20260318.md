# Task 2 实验报告：良性腺瘤 vs 非肿瘤性息肉分类

> 更新日期：2026-03-18
> 项目：基于多模态超声数据的胆囊息肉风险分级与级联分类系统（GBP-Cascade）

---

## 一、任务定义

### 1.1 级联系统中的定位

本项目采用两级级联架构：

| 级别 | 任务 | 目标 | 当前状态 |
|------|------|------|----------|
| Task 1 | 恶性 vs 非恶性 | 高召回筛查，避免漏诊恶性 | AUC 0.95+，基本达标 |
| **Task 2** | **良性腺瘤 vs 非肿瘤性息肉** | **区分需手术 vs 可观察** | **F1 ~0.55-0.63，仍需提升** |

### 1.2 Task 2 的临床意义

Task 1 筛除恶性肿瘤后，剩余患者进入 Task 2。良性腺瘤虽然非恶性，但有恶变潜能，临床上通常建议手术切除；非肿瘤性息肉（胆固醇性息肉等）一般仅需随访观察。因此 Task 2 的准确性直接影响患者是否被建议手术。

### 1.3 任务难点

- 良性腺瘤和非肿瘤性息肉在超声图像上形态高度相似，临床医生本身也经常误判
- 两者在回声、形态、基底等影像学特征上存在大量重叠
- 数据量有限（benign 仅 237 患者），视觉细粒度差异难以学到

## 四、各实验详细结果

### 4.1 五模型 Baseline 对比（3/18，统一设置）

> 实验条件：无数据增强，仅 Resize(224)+Normalize，全参数微调，CE Loss，Adam，50 epoch

| 模型 | 参数量 | 预训练 | **F1(macro)** | AUC | Acc | Benign P | Benign R | Best Ep |
|------|--------|--------|:---:|:---:|:---:|:---:|:---:|:---:|
| **Swin-T** | 27.5M | ImageNet | **0.6212** | 0.5998 | 0.7713 | 0.5000 | 0.3101 | 27 |
| UNet Encoder | 18.8M | 无（从零） | 0.5645 | **0.6467** | 0.7429 | 0.3919 | 0.2248 | 42 |
| ResNet18 | 11.2M | ImageNet | 0.5641 | 0.5406 | 0.6933 | 0.3281 | 0.3256 | 24 |
| ViT-B/16 | 85.8M | ImageNet | 0.5603 | 0.5765 | 0.7465 | 0.3971 | 0.2093 | 18 |
| VGG16 | 138.4M | ImageNet | 0.4354 | 0.3869 | 0.7713 | 0.0000 | 0.0000 | 3 |

#### 各模型训练动态分析

**ResNet18**
- 训练 Loss：0.6836 (ep1) → 0.3804 (ep10) → 0.0456 (ep50)，下降平稳
- 训练 Acc：0.7245 (ep1) → 0.8296 (ep10) → 0.9808 (ep50)
- 过拟合起点：约 epoch 14，训练 Acc 突破 93% 后测试指标不再提升
- 特点：收敛快、稳定，但容量有限，benign 召回率在 0.22-0.33 之间波动

**VGG16**
- 训练 Loss：0.8733 (ep1) → **1.3572 (ep10，发散！)** → 0.5547 (ep50)
- 训练 Acc：0.7322 (ep1) → 0.6823 (ep10，下降) → 0.7621 (ep50)
- **严重失败**：epoch 10 附近 loss 反弹，模型崩溃后缓慢恢复但始终不稳定
- benign 召回率：**全程为 0**，模型完全退化为"全部预测 no_tumor"
- 原因：138M 参数量对 1303 张训练图像严重过参数化，lr=1e-3 对此模型过高

**Swin-T**（最佳）
- 训练 Loss：0.5680 (ep1) → 0.0553 (ep10) → 0.0159 (ep50)，收敛最快
- 训练 Acc：0.7613 (ep1) → 0.9793 (ep10) → 0.9939 (ep50)
- 过拟合起点：约 epoch 6-9，但过拟合后测试指标仍缓慢提升至 epoch 27
- 特点：窗口注意力机制捕获局部纹理特征的能力强，在小数据上优于全局注意力的 ViT

**UNet Encoder**（从零训练）
- 训练 Loss：0.6059 (ep1) → 0.5444 (ep10) → 0.4964 (ep50)，**几乎不下降**
- 训练 Acc：0.7513 (ep1) → 0.7606 (ep10) → 0.7667 (ep50)
- **严重欠拟合**：无预训练权重，18.8M 参数在 1303 张图像上难以从零学习有效特征
- 但 AUC 为五模型最高（0.6467），说明其多尺度卷积特征有概率排序价值，只是分类阈值不理想
- 启示：U-Net 编码器架构有潜力，但必须引入预训练权重

**ViT-B/16**
- 训练 Loss：0.6044 (ep1) → 0.3055 (ep20) → 0.0302 (ep50)，收敛较慢
- 训练 Acc：0.7467 (ep1) → 0.8772 (ep20) → 0.9877 (ep50)
- 过拟合起点：约 epoch 18-20，之后训练 Acc 持续攀升但测试 F1 下降
- 特点：全局自注意力在小数据集上不如 Swin-T 的局部窗口注意力有效

#### 实验 B：ResNet18 + 类别权重 + 数据增强

| 项目 | 设置 |
|------|------|

- 类别权重显著提升了 benign 召回率（0.24 → 0.54），但 precision 下降

- 部分冻结 + Dropout 有效延缓了过拟合

#### 实验 C：ResNet50 + Focal Loss + WeightedSampler

| 项目 | 设置 |
|------|------|

| 采样器 | WeightedRandomSampler |
| 数据增强 | HFlip + VFlip + Rotation(15) + ColorJitter |

- Focal Loss 平衡了 precision 和 recall（0.41 / 0.40），比单纯 class weight 更均衡

- 训练 Loss 从 0.1080 稳步下降，不像 CE 那样直接塌缩到 0

### 4.3 强基线实验（3/16）

#### 实验 D：Swin-T + 患者级划分 + 强正则化

| 项目 | 设置 |
|------|------|
| 模型 | Swin-T，冻结 stage1-3，训练 stage4+head |
| 损失函数 | CE + label_smoothing=0.1 |
| 数据增强 | RandomResizedCrop + RandAugment(2,9) + HFlip + VFlip + RandomErasing |
| 正则化 | Mixup(alpha=0.2, p=0.5) + Dropout(0.3) |
| 数据划分 | **患者级** train/val/test（修复数据泄露） |
| 优化器 | AdamW, lr=2e-4, Warmup(5ep) + CosineAnnealing |
| 阈值优化 | 验证集网格搜索 [0.10, 0.90] |
| Epochs | 80 |

| 指标 | 默认阈值 (0.50) | 优化阈值 (0.54) |
|------|:---:|:---:|
| **F1 (macro)** | **0.6302** | 0.6138 |

- 训练 Loss 下降平缓：0.7110 (ep1) → 0.5980 (ep10) → 0.3728 (ep40)

- **最优出现在 epoch 9**，说明模型学习有效特征的窗口很短
- 患者级划分后指标更可靠，消除了同一患者跨集泄露的虚高

### 4.4 消融实验（3/18）

#### 实验 E / F：类别权重 vs 权重+采样器

## 五、训练动态综合分析

### 5.1 过拟合分析

| 模型 | 过拟合起始 Epoch | 最终 Train Acc | 最终 Test F1 | 过拟合程度 |
|------|:---:|:---:|:---:|------|
| VGG16 | 3 | 0.7621 | 0.4354 | 训练崩溃，完全失败 |
| ResNet18 (baseline) | 14-18 | 0.9808-1.0000 | 0.5578-0.5641 | 严重 |
| ViT-B/16 | 18-20 | 0.9877 | 0.5603 | 严重 |
| Swin-T (baseline) | 6-9 | 0.9939 | 0.6212 | 中度（过拟合后仍有提升） |
| UNet Encoder | 不存在 | 0.7667 | 0.5645 | 欠拟合（无预训练） |
| Swin-T (强基线) | 9+ | 0.8895 | 0.6302 | 轻度（正则化有效） |

**核心发现**：所有预训练模型都在 epoch 20 前达到 90%+ 训练准确率，但测试 F1 不超过 0.63。过拟合是普遍现象，根因是数据量不足（benign 仅 309 张训练图像）。

### 5.2 训练 Loss 下降模式

```
模式一：快速塌缩型（ResNet18/50, ViT）
  Loss: 0.7 → 0.3 (ep5) → 0.05 (ep15) → 0.01 (ep30) → ~0 (ep50)
  特点: Loss 迅速降至接近 0，模型记住了训练集

模式二：缓慢收敛型（Swin-T 强基线）
  Loss: 0.7 → 0.6 (ep10) → 0.4 (ep40) → 0.33 (ep80)
  特点: Label smoothing + Mixup 使 loss 无法降至 0，持续提供梯度信号

模式三：发散崩溃型（VGG16）
  Loss: 0.87 → 1.36 (ep10) → 反复震荡
  特点: 参数量过大 + 学习率过高导致优化不稳定

模式四：停滞型（UNet Encoder）
  Loss: 0.61 → 0.54 (ep10) → 0.50 (ep50)
  特点: 无预训练权重，学习进展极其缓慢
```

### 5.3 Benign 类召回率波动

benign 召回率在训练过程中呈现剧烈波动，是最不稳定的指标：

- ResNet18 augweight：epoch 6 达到 0.54，epoch 15 骤降至 0.20
- ResNet18 weightsampler：epoch 3 达到 0.49，epoch 15 降至 0.20
- Swin-T 强基线：epoch 9 达到 0.43，epoch 30 降至 0.22

**规律**：benign 召回率在训练早期较高（模型尚未过拟合到多数类时），随训练深入逐渐退化。这是类别不平衡下的典型行为——模型越训练越倾向于"安全地"预测多数类。

## 七、思考：为什么有些模型效果好，有些效果差？

### 7.1 Swin-T 为什么最好？

1. **窗口注意力机制**：Swin Transformer 将注意力限制在局部窗口内（7x7），天然适合捕捉超声图像中息肉的局部纹理特征（回声均匀性、边界清晰度）。相比之下，ViT 的全局注意力在小数据上容易分散注意力到无关区域。

2. **层级特征提取**：Swin-T 类似 CNN 的金字塔结构（从粗到细），能在不同尺度上提取特征，这对大小不一的息肉 ROI 很重要。

3. **参数效率**：27.5M 参数（中等规模），在 ImageNet 预训练后有足够容量学习域特定特征，又不至于严重过拟合。

### 7.2 VGG16 为什么完全失败？

1. **参数量严重过剩**：138.4M 参数对比 1,303 张训练图像，参数/样本比高达 106,000:1
2. **全连接层瓶颈**：VGG16 的 classifier 含两个 4096 维全连接层，这些层在小数据上极易过拟合
3. **学习率不匹配**：lr=1e-3 对 VGG 的大参数空间过于激进，导致 loss 发散
4. **架构过时**：无 BatchNorm、无残差连接，训练稳定性差

### 7.3 ViT 为什么不如 Swin-T？

1. **全局注意力的代价**：ViT-B/16 对每个 patch 计算全局注意力，在 14x14=196 个 patch 上的 O(n^2) 复杂度带来大量噪声交互
2. **位置编码的局限**：ViT 依赖固定位置编码，对 ROI 裁剪后大小各异的息肉图像不够灵活
3. **参数量大（85.8M）**：比 Swin-T 多 3 倍参数，但数据量不支持

### 7.4 UNet Encoder 为什么 AUC 高但 F1 低？

1. **AUC 高**：多尺度卷积特征（64→128→256→512→1024）提供了良好的概率排序能力，即使绝对概率值不准，相对排序是合理的
2. **F1 低**：从零训练导致特征空间未充分分离，默认阈值 0.5 下的分类决策不佳
3. **启示**：如果给 UNet Encoder 加上预训练（如使用 ResNet 编码器），性能可能显著提升

### 7.5 为什么所有模型的 Benign 召回率都很低？

**根本原因：纯图像特征不足以区分这两类**

- 超声图像中良性腺瘤和非肿瘤性息肉的视觉差异极其微妙
- 关键判别特征（息肉大小 >10mm、宽基底、血流信号等）在裁剪后的 ROI 中部分丢失
- 模型在类别不平衡下学到了"全部预测多数类也能拿到 77% 准确率"的捷径

- 引入临床结构化特征（大小、回声、蒂型等）作为显式输入
- 探索 few-shot learning 应对 benign 样本不足

### 8.4 最核心的判断

> **Task 2 的瓶颈不在模型架构或训练技巧上**（8 种模型 + 5 种损失函数 + 3 种采样策略已充分探索），**而在于纯图像信息不足以区分这两类**。最大突破口是引入超声报告的文本特征做多模态融合——Transformer 架构天然支持 Cross-Attention 图文交互，这是下一步工作的核心方向。
